Windows 下好用的软件推荐

娱乐

PotPlayer

干净强大的视频播放器，支持倍速播放（最高12倍）

工具

QTTabBar

多标签资源管理器，像浏览器标签一样同时打开管理多个文件路径

Everything

按照文件名快速搜索文件，再也不需要资源管理器慢吞吞的搜索啦

Bandizip

简洁好用的解压工具，支持自动解压，6.29版本无广告

Ditto

剪贴板历史工具，需要在不同地方来回复制文字时可以提高效率

Snipaste

超好用的截图、贴图工具，可以将截图贴在桌面上

Internet Download Manager

简称IDM，多线程下载工具，可自动嗅探网页上的资源

OBS

直播推流，视频录像工具

HoneyView

看图软件

软件开发

Notepad++

文本编辑器

Navicat

数据库客户端软件

Postman

HTTP 接口调试工具

Redis Another Desktop Manager

redis 客户端工具

Xshell

SSH 客户端软件

Typora

Markdown 编辑器

IDEA 插件

Eclipse Code Formatter

代码格式化

Alibaba Java Coding GuideLines

代码规范检查

CodeGlance

像sublime一样右侧显示代码缩略图

JRebel and XRebel

代码热部署，修改代码无需重启应用

Rainbow Bracket

彩虹色括号匹配

Save Action

代码格式化，优化导包

Tabnine

代码提示

浏览器插件

扩展管理器（Extension Manager）

一键管理所有扩展，快速开启/禁用

Adblock Plus - 免费的广告拦截器

Chrono下载管理器

SingleFile

将一个完整的页面保存到单个 HTML 文件中

Tampermonkey

油猴插件管理器，可添加各种脚本

Enhanced GitHub

显示 github 中文件大小，每个文件的下载链接

Octotree

左侧显示 github 文件树形结构

IM

Telegram

即时通讯软件（需要梯子）
